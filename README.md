# Learning_Python_Day16

Today's topic is pip and installing libraries!

**Goal:** Master Python package management and learn to install and use external libraries

- **exercise1:** Install 3 popular libraries:
  - `requests`
  - `python-dateutil`
  - `colorama`

- **exercise2:** Use `pip list` to view installed packages

- **exercise3:** Practice `pip show` to inspect package details

- **exercise4:** Create a `requirements.txt` file for your project

- **exercise5:** Uninstall and reinstall a package using pip

**mini-project:**
Colorful Console App:

- Install and use the `colorama` library  
- Create a program that prints colored text and formatted output  
- Use `python-dateutil` to parse and format dates  
- Generate a `requirements.txt` file for your project
